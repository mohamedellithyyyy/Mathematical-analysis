
# 📘 Лабораторная работа №1

Определённый интеграл Римана



### 📌 Постановка задачи

Для заданной функции на заданном отрезке необходимо:

* построить суммы Дарбу для равномерных разбиений;
* доказать интегрируемость по Риману;
* найти значение интеграла;
* проверить результат численно.

#### 📖 Аналитическая часть

В рамках лабораторной работы выполнено:

1. Построение верхних и нижних сумм Дарбу для равномерного разбиения на n частей.
2. Доказательство интегрируемости функции по Риману.
3. Вычисление предела сумм Дарбу и значения определённого интеграла.
4. Сравнение результата с формулой Ньютона–Лейбница.



#### 💻 Численные методы
